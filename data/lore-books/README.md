# TV 剧情与人物世界书恢复源

这些 TXT 是 `出包王女` 世界书剧情条目与人物条目的人工恢复源，不会被导入运行时 bundle。运行时代码按已登记的 `order`
和条目名称从真实 Tavern 世界书读取条目，并在当次生成扫描中临时启用所选条目的副本；保存于 Tavern 的原条目必须始终保持关闭。

剧情条目的硬校验只包含条目唯一、保持关闭、首尾根标签正确和标签内正文非空。正文内容与结尾措辞由世界书作者自由编辑，例如在结尾补充“不写下一集内容”不会再触发格式错误。菈菈人物条目仍保留一条宽松的姓名身份标记，避免同名错条目。

`stores/mapStore.ts` 和 `components/MapMenu.tsx`
不负责判定角色何时进入校园。菈菈只有在 EP02 三幕全部完成后才会成为校园常驻角色；地图 store 只消费位置状态，菜单只负责展示和交互。

## 第二集时间线校对

- 动画对白确认首次触碰追溯到“前天 20:43”，解除条件是婚约成立三日内再次触碰并宣告解除；不是二十四小时内证明性格不合。
- `4/8`、`4/9` 不是官方动画明示的剧中日期。TBS 页面所列 `2008/04/10` 是第 2 话播出日，不是故事日期。

  早晨开始，解除期限为 `4/10 20:43`，转学生揭晓落在 `4/11` 早晨。
- 最后一小时的谈话发生在河边。User 放弃偷袭后请菈菈出去谈，沛凯仍陪在她身边；20:43 的警报也在河边响起。
